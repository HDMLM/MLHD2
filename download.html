Getting Started

Follow these steps to install MLHD2 and start logging.

Steps

  1. Download: Get the latest launcher from the download page.
  2. Install: Run the installer. If Windows SmartScreen warns you, select "More info" then "Run anyway" while we work on code-signing (it's really expensive dude...).
  3. Launch: Start MLHD2 alongside Helldivers 2.
  4. Log missions: After each mission, record your results.
  5. Export: Generate a summary to share on Discord or analyze in your favorite tool.

FAQ

Does MLHD2 read or modify game files?

No. MLHD2 is a separate logging app and avoids touching game files.

Do I have to share to Discord?

Partially. For transparency, due to the nature of this program, the Discord integration is nested rather deep into our code, there is an option to disable public display.
This will then only send the log to a private server only the team can access simply to ensure the program has recognised that the log has been sent correctly.
However for your own use, you are not forced to provide a webhook URL to the program to send to.

Where are my logs stored?

Logs are stored locally on your machine. You choose what to export or share.
